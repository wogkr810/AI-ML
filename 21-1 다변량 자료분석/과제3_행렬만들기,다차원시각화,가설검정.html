문제1 : 공분산행렬으로 부터 상관계수 행렬 만들기

수식 : Σ=V-1/2ρV-1/2

공분산행렬(Σ)

sigma<-matrix(c(4,1,2,1,9,-3,2,-3,25),nrow=3)
sigma
##      [,1] [,2] [,3]
## [1,]    4    1    2
## [2,]    1    9   -3
## [3,]    2   -3   25

대각성분행렬(V-1/2)

v_1_2<-diag(3)
for (i in 1:nrow(v_1_2)){
  v_1_2[i,i]<-(1/sigma[i,i])^(1/2)
}
v_1_2
##      [,1]      [,2] [,3]
## [1,]  0.5 0.0000000  0.0
## [2,]  0.0 0.3333333  0.0
## [3,]  0.0 0.0000000  0.2

상관계수행렬(ρ)

low<-v_1_2%*%sigma%*%v_1_2
low
##           [,1]       [,2] [,3]
## [1,] 1.0000000  0.1666667  0.2
## [2,] 0.1666667  1.0000000 -0.2
## [3,] 0.2000000 -0.2000000  1.0

참고: cov2cor() 코드로도 가능

x<-cov2cor(sigma)
x
##           [,1]       [,2] [,3]
## [1,] 1.0000000  0.1666667  0.2
## [2,] 0.1666667  1.0000000 -0.2
## [3,] 0.2000000 -0.2000000  1.0
x==low
##      [,1] [,2] [,3]
## [1,] TRUE TRUE TRUE
## [2,] TRUE TRUE TRUE
## [3,] TRUE TRUE TRUE

–> 결과 값 같은 것 확인 가능!

가설설정

H0 : 법이 제정되기 전 과 후의 운전자 사망자의 수가 차이가 없다.

Set 유의수준(α) = 5 %
